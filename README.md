# Aditya Narvekar - Portfolio Website

This is the source code for my professional portfolio website at [www.adityanarvekar.com](https://www.adityanarvekar.com).

## Built With

- **Hugo** - Fast static site generator
- **GitHub Pages** - Free hosting
- **GitHub Actions** - Automatic deployment

## Quick Start

### Editing Content on GitHub.com

The easiest way to update your portfolio:

1. Go to your repository on GitHub.com
2. Navigate to the `content/` folder
3. Click on the file you want to edit (e.g., `_index.md`)
4. Click the pencil icon to edit
5. Make your changes
6. Click "Commit changes" at the bottom
7. Your site will automatically rebuild and deploy in ~1 minute!

### Local Development

If you want to preview changes locally before publishing:

```bash
# Start the development server
hugo server

# View your site at http://localhost:1313
```

## Folder Structure

```
├── content/              ← YOUR CONTENT GOES HERE
│   ├── _index.md        ← Home page content
│   ├── work/            ← Work samples
│   └── writing/         ← Articles
├── static/              ← Images and PDFs
│   └── images/          ← Work preview images
├── themes/              ← Site design (don't edit unless you want to change design)
└── hugo.toml            ← Site configuration
```
